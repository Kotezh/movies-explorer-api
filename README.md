## Запуск проекта

* Клонировать репозиторий

`git clone https://github.com/Kotezh/movies-explorer-api.git`

* Установить зависимости

  `npm install`

* Запустить проект

  `npm start` или `npm run dev` (с hot-reload)

* Порт для просмотра результатов в браузере

  `http://localhost:3000`
